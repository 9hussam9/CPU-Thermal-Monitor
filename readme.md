# CPU Thermal Monitor (Digital Twin)

A cross-platform CPU monitoring tool built with Python and Flet. This project uses a Digital Twin model to estimate thermal output based on processor load, bypassing Windows hardware access restrictions.

## Requirements
- Python 3.10 or higher
- Flet Framework

To install the necessary dependencies, run:

pip install flet --break-system-packages
